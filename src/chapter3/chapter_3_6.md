# Dictionaries

Dictionaries are key-value pairs. Values can be assigned and accessed using [] brakets.
ss
```python
dict_ = {'apples': 10, 'oranges':15}
print(dict_)
print(dict_['apples'])
dict_['apples'] = 20
print(dict_['apples'])
print(dict_.keys())
print(dict_.values())
```
Output
```
{'apples': 10, 'oranges': 15}
10
20
dict_keys(['apples', 'oranges'])
dict_values([20, 15])
```
